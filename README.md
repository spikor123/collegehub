# 🎓 CollegeHub — Your Campus, Connected

![CollegeHub Banner](https://images.unsplash.com/photo-1541339907198-e08756ebafe3?q=80&w=2070&auto=format&fit=crop)

CollegeHub is a modern, digital ecosystem designed specifically for Indian universities. It centralizes all campus activities into a single, beautiful, and secure platform.

## 🚀 Live Demo
**Check out the live deployment here:** [https://spikor123.github.io/collegehub/](https://spikor123.github.io/collegehub/)

---

## ✨ Key Features

### 📢 Smart Notice Board
Real-time academic and campus announcements with priority tagging and categorization.

### 📅 Events & Workshops
Discovery portal for college fests, hackathons, and seminars with integrated registration.

### 📝 Study Notes Repository
Securely shared handwritten and digital study material organized by department (CSE, EEE, ME, etc.) and semester.

### 🍱 Peer-to-Peer Marketplace
A secure, campus-only marketplace for buying and selling textbooks, gadgets, and notes.

### 🗳️ Anonymous Campus Polls
A platform for students to voice their opinions on campus decisions with verified, anonymous voting.

### 👩‍🏫 Faculty & Admin Portals
- **Attendance Management:** Modern, high-contrast digital registers with progress tracking for students and quick-marking tools for teachers.
- **Student Data:** Secure directory with academic metrics.
- **Admin Dashboard:** System-wide controls, banner customization, and audit logs.

### 🔔 Integrated Notifications
A unified notification system using the Web Notification API for real-time alerts on notices, assignments, and campus events.

## 🛠️ Technology Stack
- **Framework:** [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS 4.0](https://tailwindcss.com/)
- **Components:** [Radix UI](https://www.radix-ui.com/) & [Lucide Icons](https://lucide.dev/)
- **State Management:** [Zustand](https://zustand-demo.pmnd.rs/)
- **Database:** [Supabase](https://supabase.com/) (Mocked in demo mode)

## 🏃‍♂️ Getting Started (Local Development)

```bash
# Install dependencies
npm install

# Run the development server
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to see the application running locally.

## 📦 Deployment
The application is configured for **Static Export** (`output: 'export'`) and is natively deployed via GitHub Actions to GitHub Pages.
